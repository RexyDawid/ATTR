ATTR Antivirus - README

Willkommen beim ATTR Antivirus!

Beschreibung:
-----------------
ATTR Antivirus ist ein Antivirus-Tool, das speziell für Linux-Systeme entwickelt wurde.
Es bietet Funktionen wie das Sichern von Daten und das Scannen nach möglichen Bedrohungen.

Funktionsweise:
-----------------
1. Sichere meine Daten:
   - Dieser Button erstellt einen "Safe-Ordner" auf deinem System, in dem du wichtige Dateien sichern kannst.

2. PC durchsuchen:
   - Führt einen Scan auf deinem PC durch, um nach Bedrohungen zu suchen. Der Scan dauert in der Regel 3 bis 5 Minuten.

3. Exit:
   - Verlassen des Programms.

Voraussetzungen:
-----------------
- Stelle sicher, dass die Datei "Linux.dll" im "Update/"-Ordner vorhanden ist.
- Die Datei "Connect.dll" wird benötigt, um eine Verbindung zum Linux-System herzustellen.

Fehlerbehebung:
-----------------
- Wenn eine der benötigten Dateien fehlt, wird eine Fehlermeldung angezeigt, die erklärt, was genau fehlt.

Verwendung:
-----------------
- Um das Programm zu starten, führe die "Host.bat"-Datei aus.
- Stelle sicher, dass alle Dateien im richtigen Verzeichnis liegen.
